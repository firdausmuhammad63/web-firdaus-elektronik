# Toko Elektronik Firdaus

Website resmi **Toko Elektronik Firdaus**, menampilkan informasi produk elektronik, promo, kontak, dan lokasi toko.

## Deskripsi
Website ini dibuat menggunakan **HTML**, **CSS (Bootstrap 5)**, dan **JavaScript** untuk interaktivitas dasar seperti:
- Hamburger toggle pada navbar
- Pencarian produk
- Form kontak dengan simulasi toast notifikasi
- Tombol WhatsApp mengarah langsung ke chat toko

Website bersifat **responsive**, sehingga nyaman diakses dari desktop maupun perangkat mobile.

## Fitur
- **Beranda**: Informasi umum toko dan highlight produk/promo.
- **Produk**: Daftar produk elektronik yang dijual.
- **Promo**: Informasi promo dan diskon.
- **Tentang**: Profil dan sejarah toko.
- **Kontak**: Form untuk mengirim pesan, alamat, nomor telepon/WhatsApp, dan peta lokasi.
- **Favicon dan meta description**: Optimalisasi tampilan di browser dan SEO dasar.
- **WhatsApp Toggle**: Tombol mengambang untuk chat langsung.

## Teknologi
- **HTML5 & CSS3**
- **Bootstrap 5**
- **Bootstrap Icons**
- **JavaScript ES6**
- **Google Maps Embed**
- **GitHub Pages** untuk hosting (opsional)
